使用说明

  1. shadowsocks 原理是借助国外的 VPS 翻墙。国外的 VPS 上运行服务端，在本地运行本地端，就可以建立加密连接。如果你还没有 VPS，末尾列举了一些常见的 VPS 提供商。
  2. 用文本编辑器编辑 config.json，请注意一定要修改密码。
    server 你的 VPS 的地址
server_port 使用的服务端口
local_port 本地 socks5 端口
password 请设置一个你的密码
  3. 用 tools 目录里的 winscp 上传 server.py 和 config.json 到 VPS 上的某个目录。如果是 root 用户，放到 /root/ 下；如果是其他用户，放到 /home/用户名 目录下。
  4. 用 tools 目录里的 putty 登录到 VPS。
    安装 python：
    Debian, Ubuntu:
    sudo apt-get install python
    CentOS:
    yum install python
    cd 进入到上传的目录，运行（注意后面的 &）：
    nohup python server.py > log &
    退出 putty。接下来只要 VPS 没有重启，服务端会一直运行下去。
  5. 运行 local.exe
  6. 修改浏览器的代理设置。使用 SOCKS5 代理，地址为刚才设置的本地 socks5 端口（默认 127.0.0.1:1080）。推荐 Firefox 配合 Autoproxy 使用，Chrome 配合 SwitchySharp 使用。

VPS 推荐列表：

  1. 知名，稳定，速度快，比较贵，适合多人合租（推荐）：
    Lunarpages, Linode, Rackspace
  2. 不知名，便宜，偶尔不稳定，适合个人用：
    photonvps, buyvm, virpus, yardvps, vexxhost
  3. 亚马逊 EC2 可以免费用一年，但是注册和设置比较复杂：
    EC2
  4. 另外有一些免费的 VPS，可以 Google 搜索 free vps，不过相对收费的来说会有各种问题。
    推荐选择 Xen 或者 KVM 的 VPS，尽量不要选择 OpenVZ。
    有关如何设置 VPS 可以 Google 如何设置 linode vps ，如何申请免费 EC2 之类的关键词。
